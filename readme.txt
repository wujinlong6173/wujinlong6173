项目名称：三层网络管理系统 L3NMS
项目功能：管理物理路由器、虚拟路由器混合组成的三层网络，例如BGP&MPLS L3VPN，数据中心的虚拟路由器。
准备解决的问题：
  BPG&MPLS L3VPN技术诞生之后，几大标准组织都定义了业务模型，但是，这些业务模型存在较大差异，而且臃肿复杂。
  几大厂商都实现了管理系统，但是，每个系统只能管理本厂商的设备，管理其它厂商的设备时，适配工作巨大，而且功能大打折扣。
  企业从运营商购买L3VPN后，可以执行的私网配置非常有限，受制于厂商提供的管理系统；没法统一管理全网的物理部分和虚拟部分。
解决思路：
  企业从运营商购买L3VPN，不要作为一个整体，而是拆分成虚拟路由器、和虚拟链路。
  运营商出售虚拟路由器后，提供一个网络链接，打开该链接即可对虚拟路由器进行配置，例如，设置IP地址，配置OSPF协议，配置静态路由。
  配置方式可以是命令行、NetConf等，最重要的是 -- 操作人员感觉不到这是一台虚拟路由器，还是一台物理路由器。
  于是，企业用来管理物理网络的系统，自然就能管理购买的L3VPN，只需要适配一款新设备类型。

中文乱码的问题：
1. Putty客户端中文乱码的解决办法，打开Putty Configuration > Window > Translation，将Remote character set修改为Use font encoding。再切换到Session，将设置保存为Default Settings。
2. 拓扑界面中文乱码的解决办法，启动本程序时，设置JVM参数 -Dfile.encoding=gbk。
